# ScoutIQ – Football Intelligence Platform

ScoutIQ is a data-driven football analytics dashboard built with Python and Streamlit.

The application allows users to explore player data, analyze performance metrics, and simulate transfer impacts using football data.

## Features

- Player scouting dashboard
- Market efficiency analysis
- AI player profiling using clustering
- Transfer impact simulator
- Club ranking based on squad strength

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn
